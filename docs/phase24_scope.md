# Phase 24 scope

Phase 24 adds inferential comparisons of homologous selected-model parameters between groups.

Included:

- pairwise comparisons for each model and parameter combination;
- bootstrap confidence intervals for mean differences;
- permutation p-values;
- Hedges' g and Cliff's delta effect sizes;
- Benjamini-Hochberg multiplicity correction;
- configurable minimum group size and error continuation;
- plotting, export, tests, documentation, and example.

Not included:

- pooling parameters from different constitutive models;
- mixed-effects or repeated-measures models;
- covariate adjustment;
- automatic biological interpretation;
- causal claims from group differences.
