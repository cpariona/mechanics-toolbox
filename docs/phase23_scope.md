# Phase 23 scope

Phase 23 extracts fitted parameters from the model selected independently for each specimen and produces population-level descriptive summaries.

Included:

- long-form selected-parameter extraction;
- retention of specimen and group identifiers;
- retention of bootstrap parameter intervals when available;
- overall summaries by model and parameter;
- group summaries by model and parameter;
- extraction-error capture;
- plotting, export, tests, documentation, and example.

Not included:

- pooling parameters from different model families;
- inferential group comparisons;
- mixed-effects modeling;
- multiplicity correction;
- automatic biological interpretation.
