# Phase 18 scope

## Included

- residual diagnostics for one completed constitutive fit;
- lag-one autocorrelation screening;
- residual trend screening against deformation;
- residual-magnitude screening against predicted stress;
- standardized-residual outlier flags;
- plotting and CSV/MAT export;
- tests, documentation, and a runnable example.

## Not included

- automatic observation removal;
- formal time-series hypothesis tests;
- weighted or generalized least-squares refitting;
- model replacement or automatic model selection based on residual flags;
- uncertainty propagation from specimen geometry or preprocessing.
