# Phase 17 scope

Phase 17 adds one-model parameter-stability analysis across nested deformation
windows.

Included:

- configurable maximum-deformation fractions;
- repeated bounded fitting with existing fitting contracts;
- per-window fit status and metrics;
- per-parameter relative-range stability flags;
- plotting, export, tests, documentation, and an example.

Excluded:

- automatic model replacement;
- profile likelihood;
- specimen-level uncertainty propagation;
- optimization of the experimental deformation range.
