# Phase 19 scope

Phase 19 adds an integrated reliability summary for one constitutive fit.

Included:

- convergence and fit-quality checks;
- bootstrap-success screening;
- identifiability, deformation-window stability, and residual flags;
- configurable reliability classification;
- plotting, export, tests, documentation, and example.

Excluded:

- automatic model rejection;
- automatic data removal or refitting;
- uncertainty propagation from specimen geometry;
- formal hypothesis testing or Bayesian model evidence.
