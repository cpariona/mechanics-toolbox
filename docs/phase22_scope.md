# Phase 22 scope

Phase 22 adds reliability-aware constitutive model comparison across multiple specimens.

Included:

- independent Phase 21 comparison for each specimen;
- specimen-level selected-model summary;
- aggregate model-selection frequencies;
- optional group-level frequencies;
- configurable continuation after specimen failures;
- plotting, export, tests, documentation, and example.

Not included:

- pooled fitting across specimens;
- mixed-effects constitutive models;
- inferential comparison of selection frequencies;
- automatic group hypothesis testing;
- replacement of specimen-level diagnostic review.
