# Phase 20 scope

Phase 20 adds a single reproducible workflow for one constitutive fit and its
reliability diagnostics.

Included:

- constitutive fitting;
- optional residual bootstrap uncertainty;
- optional identifiability analysis;
- optional deformation-window stability analysis;
- optional residual diagnostics;
- integrated reliability classification;
- diagnostic-error capture;
- consolidated export;
- tests, documentation, and example.

Not included:

- automatic model selection across multiple constitutive models;
- automatic exclusion of observations;
- automatic modification of fitting bounds;
- batch execution across specimens;
- statistical comparison between groups.
