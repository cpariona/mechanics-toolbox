# Phase 25 scope

Phase 25 integrates model selection, selected-parameter population summaries, and inferential group comparisons into a final study report.

Included:

- Markdown report generation;
- model-selection summary;
- selected-model parameter summaries;
- group-level descriptive summaries;
- inferential comparison table with effect sizes and adjusted p-values;
- standard report figures;
- configurable figure inclusion and export settings;
- tests, documentation, and example.

Not included:

- rerunning extraction, fitting, diagnostics, or inference;
- automatic scientific conclusions;
- document formats other than Markdown;
- mixed-effects or hierarchical constitutive modeling;
- replacement of specimen-level diagnostic review.
